# misc-ui

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```
### 打包组件库
```
yarn run lib
```

### 发布到npm
先修改package.json的version
```
npm publish
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### Run your unit tests
```
yarn run test:unit
```
